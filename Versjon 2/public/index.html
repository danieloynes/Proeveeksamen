Om Dyreklinikk Administrasjonssystem

Dette er et enkelt administrasjonssystem for dyreklinikker som lar deg holde oversikt over eiere og deres dyr.

Systemet bruker:

  • HTML, CSS og JavaScript for brukergrensesnittet
  • Node.js med Express for backend
  • MySQL for databasen
